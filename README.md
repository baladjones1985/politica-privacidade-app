# Política de Privacidade

**Última atualização:** 23 de dezembro de 2025

## 1. Introdução

Esta Política de Privacidade descreve como o aplicativo **Mabe Demonstrativo** ("nós", "nosso" ou "Aplicativo") coleta, usa, armazena e protege as informações dos usuários ("você" ou "usuário").

Ao utilizar nosso Aplicativo, você concorda com as práticas descritas nesta Política de Privacidade.

## 2. Informações que Coletamos

### 2.1. Informações Fornecidas pelo Usuário
- **Dados de clientes:** Nome, telefone, endereço, CPF/CNPJ e outras informações cadastradas pelo usuário para criação de orçamentos, recibos e contratos.
- **Dados de serviços e materiais:** Descrições, valores e quantidades de itens cadastrados.
- **Dados da empresa:** Informações da empresa do usuário, como razão social, CNPJ, endereço e dados bancários para geração de documentos.

### 2.2. Informações Coletadas Automaticamente
- **Identificador do dispositivo:** Para fins de licenciamento e funcionamento do aplicativo.
- **Informações do dispositivo:** Sistema operacional, modelo do dispositivo e versão do aplicativo.

### 2.3. Permissões do Aplicativo
O aplicativo pode solicitar as seguintes permissões:
- **Acesso à Internet:** Para sincronização de dados e verificação de licença.
- **Acesso aos Contatos:** Para facilitar o preenchimento de dados de clientes (opcional).
- **Armazenamento:** Para salvar e exportar documentos PDF.

## 3. Como Usamos Suas Informações

Utilizamos as informações coletadas para:
- Gerar orçamentos, recibos, contratos e outros documentos comerciais.
- Armazenar e organizar dados de clientes e serviços.
- Sincronizar dados entre dispositivos (quando aplicável).
- Verificar a licença e autenticidade do aplicativo.
- Melhorar a experiência do usuário e o funcionamento do aplicativo.

## 4. Armazenamento de Dados

### 4.1. Armazenamento Local
- Os dados são armazenados principalmente no dispositivo do usuário.
- Utilizamos banco de dados local (SQLite) para garantir acesso offline.

### 4.2. Armazenamento em Nuvem
- Quando habilitada a sincronização, os dados podem ser armazenados em servidores seguros (Supabase).
- Os dados são transmitidos de forma criptografada (HTTPS/TLS).

## 5. Compartilhamento de Informações

**Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros**, exceto:
- Quando necessário para o funcionamento do serviço (ex: serviços de hospedagem em nuvem).
- Quando exigido por lei ou ordem judicial.
- Com seu consentimento expresso.

## 6. Segurança dos Dados

Implementamos medidas de segurança para proteger suas informações:
- Criptografia de dados em trânsito (HTTPS/TLS).
- Armazenamento seguro em banco de dados.
- Controle de acesso aos dados.

No entanto, nenhum método de transmissão ou armazenamento eletrônico é 100% seguro, e não podemos garantir segurança absoluta.

## 7. Seus Direitos

Você tem o direito de:
- **Acessar** seus dados pessoais armazenados no aplicativo.
- **Corrigir** dados incorretos ou desatualizados.
- **Excluir** seus dados do aplicativo.
- **Exportar** seus dados em formato utilizável.
- **Revogar** o consentimento para uso de dados.

Para exercer esses direitos, entre em contato conosco através do WhatsApp: **(66) 99662-6207**.

## 8. Retenção de Dados

- Seus dados são mantidos enquanto você utilizar o aplicativo.
- Ao desinstalar o aplicativo, os dados locais são removidos do dispositivo.
- Dados sincronizados em nuvem podem ser excluídos mediante solicitação.

## 9. Uso por Menores

Este aplicativo não é destinado a menores de 18 anos. Não coletamos intencionalmente informações de menores.

## 10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas através do próprio aplicativo ou por outros meios apropriados.
